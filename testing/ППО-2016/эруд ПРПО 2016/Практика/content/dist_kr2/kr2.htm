Контрольная работа №2. Использование Android NDK для написания кода на С\C++.

 

Цель контрольной  работы.

     Изучить взаимодействие Java и С++ через связь посредством JNI\NDK; преобразования параметров; генерацию заголовочного файла для нативного кода на С++; выполнить компиляция файлов библиотек на стороне С; научится создавать и использовать файлы проекта Android.mk и Application.mk 

Указания по выбору варианта контрольной работы.

Соответствующий вариант работы выбирается по правилу: последняя цифра номера зачетной книжки + 1.

 

Задания

Разработать C++ функцию и подключить ее в Android приложение. Исходные данные передаются со стороны Java. На стороне С\С++ вычисляется функция:

Вариант 1. Со стороны Java на сторону С передается массив строк. Обратно возвращается массив целых, который содержит вычисленные длины строк.
Вариант 2. На сторону С передается массив строк, после чего выполняется сортировка строк по какому-либо критерию.

Вариант 3.  Передается 2-мерный массив целых. Возвратить массив  такой же размерности с инвертацией по строкам. 

Вариант 4. Передается 2-мерный массив целых. Возвратить массив такой же размерности с инвертацией по столбцам. 

Вариант 5.  Передается матрица целых размерностью NxN. Инвертировать по основной диагонали.

Вариант 6.   Передается матрица чисел с плавающей точкой размерностью NxN. Инвертировать по основной диагонали.

Вариант 7.  Передается массив строк. Строки инвертировать.

Вариант 8.  Передается строка. Возвратить массив пар {символ, частота встречаемости символа в исходной строке}

Вариант 9.  Реализовать на стороне С функцию, выполняющую шифровку\дешифровку текстового сообщения. Алгоритм любой.

Вариант 10.  На сторону С приходит строка в виде имени текстового файла. Файл открыть, прочитать, результат возвратить в Java как текстовый массив.
